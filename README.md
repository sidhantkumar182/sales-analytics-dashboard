# Sales Analytics Dashboard

## Overview

A data analytics project built using Python, Pandas, SQL, and Excel to analyze sales performance across products and regions.

## Features

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Revenue and Profit Analysis
* Region-wise Sales Insights
* Product Performance Tracking
* SQL-based Reporting

## Technologies Used

* Python
* Pandas
* SQL
* Excel
* Matplotlib

## Key Insights

* Identified top-performing products.
* Analyzed regional sales trends.
* Generated KPI-based reports for business decision making.

## Future Improvements

* Interactive Power BI Dashboard
* PostgreSQL Integration
* Automated ETL Pipeline
